# Changelog

## 0.3.0 - Cloud Tool Catalog

- 工具中心本体与具体工具脚本解耦，安装脚本不再把 `tool/` 复制为本地已安装工具。
- 新增云端工具检测、云端预览、按需安装和本地卸载。
- fzf 顶部常驻 Logo、版本、路径和快捷键说明。
- 本地已安装工具与云端可安装工具合并展示，云端列表自动跳过已安装项。
- 更新逻辑兼容旧安装用户，保留本地已安装工具，不再从应用包同步 `tool/`。

## 0.2.0 - Linux Tool Center

- 重构为本地 Linux 终端应用。
- 新增 `bin/linux-tool` 主程序。
- 新增 `lib/` 公共模块：工具加载、fzf UI、预览、Logo、日志、安全确认、更新和版本。
- 工具统一从 `tool/*.sh` 自动发现。
- 安装路径切换到当前用户的 `~/.local` 目录，不再强制 root。
